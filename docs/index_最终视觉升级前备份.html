建筑结构安全计算参数

当前版本使用轻量结构安全计算引擎，可对老旧农房、砖混结构、钢筋混凝土建筑进行快速安全评估；后续可接入 ABAQUS 工业求解器。


参数说明

结构类型

选择建筑主体结构形式，例如砖混结构、钢筋混凝土结构、钢结构、木结构/老旧农房。不同结构的抗震、承载和老化风险不同。

建筑层数

表示建筑总层数。层数越高，结构受力越复杂，墙体、梁板和基础承受压力更大，风险评分会相应提高。

建筑年限

表示建筑已使用年数。年限越长，可能存在材料老化、墙体开裂、构件耐久性下降、基础沉降等问题。

最大裂缝宽度 mm

表示现场观测到的最大裂缝宽度，单位为毫米。裂缝越宽，说明墙体、梁板或结构连接部位可能存在更明显的受力异常。

附加荷载系数

表示建筑额外承受的压力大小。1.0 为正常荷载，1.2 为略高，1.4 为较高，1.6 以上通常需要重点关注。

地基情况

表示基础状态。若存在疑似沉降，可能出现墙体斜裂缝、门窗变形、地面下沉、房屋局部倾斜等风险。

系统定位

本模块不是简单展示图，而是根据结构类型、建筑年限、裂缝宽度、荷载系数、地基情况进行轻量化安全计算， 输出风险评分、应力比、最大位移估算、安全系数和整改建议。后续可通过 Abaqus Python 接口升级为工业级有限元求解。
